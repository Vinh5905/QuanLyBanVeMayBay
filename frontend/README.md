# Frontend (React + Vite)

## Setup

```bash
npm install
npm run dev
```

## Environment

Create `.env`:

```
VITE_API_BASE_URL=Base URL of the backend API```

## Scripts

* `npm run dev` ‚Äî start dev server
* `npm run build` ‚Äî build project
* `npm run format` ‚Äî format code

## Structure

* `src/api` ‚Äî API calls & adapters
* `src/app` ‚Äî router, providers
* `src/pages` ‚Äî pages
* `src/components` ‚Äî reusable UI

## Notes

* Uses Axios for HTTP
* Uses React Router for navigation
* API responses follow BE-09 format
